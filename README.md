# CryptoLovec

Данный скрипт написан на Telethon и автоматически собирает чеки из чатов на вашем аккаунте.

# Функционал:
1. Автосбор чеков.
2. Если в сообщении есть пароль, использует его для сбора.
3. Анти-каптча.

# Установка (Termux):
1. apt update
2. apt upgrade
3. apt install git
4. apt install python
5. git clone https://github.com/CryptoLovec/cryptolovec2
6. pip install shutil
7. pip install pycryptodome
8. pip install telethon
9. pip install requests
10. cd cryptolovec2
11. python main.py
